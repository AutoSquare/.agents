# 论文摘要

## 适用范围

用于把论文转换为可复用的中文研究笔记。优先读取用户提供的 PDF、Zotero 条目、摘要或 MCP 检索结果。

## 单篇论文流程

1. 记录完整元数据：作者、年份、题名、期刊或会议、DOI、链接。
2. 提取研究问题、理论框架、方法、数据、主要发现。
3. 区分作者结论与模型推断，不扩大结论适用范围。
4. 标出局限、适用场景、与用户主题的关系。
5. 给出可引用观点，但不编造页码。

## 多篇论文流程

1. 先按主题、方法或结论分组。
2. 比较共识、差异与证据强弱。
3. 标记相互引用、共同数据源或重复研究团队。

## 输出格式

```markdown
## 文献信息
- 作者：
- 年份：
- 题名：
- 来源：
- DOI/链接：

## 结构化摘要
- 研究问题：
- 方法与数据：
- 主要结论：
- 局限：
- 与当前主题的关系：

## 可引用观点
- 观点：
  - 依据：
  - 使用场景：
```

## 约束

- 未读取全文时必须标注“基于摘要/元数据”。
- 不把预印本等同于同行评审论文。
- 不虚构页码、图表编号或精确统计值。
