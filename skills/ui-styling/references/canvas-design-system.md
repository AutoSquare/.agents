# Canvas 设计系统

视觉设计哲学、系统化构图与精密视觉传达。

## 设计哲学方法

Canvas 设计采用两阶段流程：

### 阶段一：设计哲学创作

建立视觉哲学——通过形态、空间、色彩与构图表达的美学运动。不是布局或模板，而是纯粹的视觉哲学。

**产出内容：** 设计宣言，强调：
- 视觉表达优先于文字
- 空间传达
- 艺术化诠释
- 文字极简，作为视觉点缀

**哲学结构（4–6 段）：**
- 空间与形态原则
- 色彩与材质方法
- 尺度与节奏指引
- 构图与平衡规则
- 视觉层级体系

### 阶段二：视觉表达

通过 Canvas 作品表达哲学：
- 90% 视觉设计
- 10% 必要文字
- 博物馆级执行标准
- 系统化模式
- 精密构图

## 核心原则

### 1. 视觉传达优先

信息存在于设计中，而非段落里。通过以下方式表达思想：
- 色块与色域
- 几何精度
- 空间关系
- 视觉重量与张力
- 形态与结构

### 2. 极简文字融入

文字作为罕见而有力的手势：
- 绝不用段落
- 仅保留必要词句
- 融入视觉架构
- 小标签，大影响
- 排版作为视觉元素

### 3. 专业工艺

作品应呈现：
- 精心打磨
- 倾注心力
- 仿佛历经无数小时
- 来自领域顶尖水准
- 大师级执行

### 4. 系统化模式

使用科学的视觉语言：
- 重复模式
- 完美形状
- 密集标记累积
- 分层元素
- 耐心重复，值得细读

## 设计运动示例

### Concrete Poetry（具体诗）
**哲学：** 通过纪念碑式形态与大胆几何传达信息。

**表达：**
- 大面积色块
- 雕塑感排版（巨大文字、微小标签）
- 粗野主义空间分割
- 波兰海报能量 × Le Corbusier
- 通过视觉重量与空间张力传递思想
- 文字作为罕见而有力的手势

### Chromatic Language（色彩语言）
**哲学：** 色彩作为主要信息系统。

**表达：**
- 几何精度
- 色域承载意义
- 排版极简——小型 sans-serif 标签
- Josef Albers 色彩互动 × 数据可视化
- 信息与意义编码于空间与色彩
- 文字仅锚定色彩已展示的内容

### Analog Meditation（模拟冥想）
**哲学：** 通过质感与留白实现安静的视觉沉思。

**表达：**
- 纸纹、墨迹渗透
- 大量负空间
- 摄影与插画占主导
- 排版低语（小字号、克制）
- 日式摄影书美学
- 图像跨页呼吸
- 文字极少出现——仅短句

### Organic Systems（有机系统）
**哲学：** 自然聚类与模块化生长模式。

**表达：**
- 圆润形态
- 有机排列
- 从自然到建筑的色彩
- 通过视觉图表传递信息
- 空间关系与图标
- 文字仅用于漂浮于空间的关键标签
- 构图通过空间编排讲述故事

### Geometric Silence（几何静默）
**哲学：** 纯粹秩序与克制。

**表达：**
- 基于网格的精度
- 大胆摄影或鲜明图形
- 戏剧性负空间
- 排版精确而极简
- 小量必要文字，大面积安静区域
- 瑞士形式主义 × 粗野主义材质诚实
- 结构传达，而非文字
- 每一处对齐都经无数微调

## 实施指南

### 隐性引用融入

嵌入概念内核，但不刻意宣告：
- 小众引用无形编织
- 知情者直觉感知
- 其他人体验大师级抽象构图
- 如爵士乐手隐式引用另一首曲目
- 精妙含蓄，从不字面化
- 引用安静增强深度

### 色彩方法

**有意图的调色板：**
- 限定颜色（2–5 种）
- 统一协调的体系
- 有目的的色彩关系
- oklch 色彩空间保证精度
- 每个色阶承载意义

**示例调色板：**
```
--color-primary: oklch(0.55 0.22 264)
--color-accent: oklch(0.75 0.18 45)
--color-neutral: oklch(0.90 0.02 264)
--color-dark: oklch(0.25 0.15 264)
```

### 排版体系

**偏好细字重：**
- 轻字重（200–300）
- 干净 sans-serif
- 几何精度
- 标签用小字号
- 冲击时刻用大字号

**字体整合：**
- 搜索 `./canvas-fonts` 目录
- 下载所需字体
- 将排版带入 Canvas
- 作为艺术的一部分，而非数字排版

### 构图规则

**系统化方法：**
- 重复模式建立节奏
- 完美几何形状
- 严谨克制的排版
- 参考标记暗示虚构的学科规范
- 密集累积构建意义
- 分层模式奖励细读

**间距纪律：**
- 元素不得超出页面
- 元素不得重叠
- 每个元素在 Canvas 边界内
- 边距不可妥协
- 留白与清晰分隔
- 专业执行是硬性要求

### Canvas 边界

**技术规格：**
- 默认单页（按需多页）
- PDF 或 PNG 输出
- 高分辨率
- 干净边距
- 边界内收束的构图
- 格式无瑕

## 多页设计系统

创建多页作品时：

### 方法
- 将首页视为咖啡桌书籍中的单页
- 沿同一哲学扩展更多页面
- 各具特色但统一协调
- 页面得体地讲述故事
- 充分创作自由

### 一致性元素
- 共享调色板
- 一致的排版体系
- 相关的构图方法
- 视觉语言连续性
- 贯穿的哲学线索

### 变化策略
- 每页独特变化
- 不同视觉焦点
- 多样的空间排列
- 互补模式
- 渐进式视觉叙事

## 执行清单

定稿前确认：

- [ ] 哲学指导每个决策
- [ ] 最多 90% 视觉、10% 文字
- [ ] 文字极简且已融入
- [ ] 无重叠或超出页面
- [ ] 边距与间距洁净无瑕
- [ ] 构图与整体艺术统一协调
- [ ] 呈现精心打磨感
- [ ] 可见大师级执行
- [ ] 精妙高级，绝不业余
- [ ] 可陈列于博物馆
- [ ] 证明无可辩驳的专业性
- [ ] 格式无瑕
- [ ] 每个细节完美

## 质量标准

### 应避免
- 卡通化美学
- 业余执行
- 文字过重构图
- 随意放置
- 元素重叠
- 间距不一致
- 明显 AI 生成感
- 缺乏精修

### 应达成
- 博物馆级品质
- 杂志级水准
- 艺术对象地位
- 仿佛历经无数小时
- 领域顶尖工艺
- 哲学一致连贯
- 视觉精妙高级
- 系统化精确

## 精修流程

### 初稿
基于哲学与原则创作。

### 第二遍（关键）
- 不要增加更多图形
- 精修已有内容
- 使极其清晰锐利
- 尊重极简主义哲学
- 增强与整体艺术的统一性
- 让现有元素更具艺术性
- 打磨而非扩展

### 最终验证
用户已说：「还不够完美。必须洁净无瑕，是工艺杰作，仿佛即将陈列于博物馆。」

交付前按此标准执行。

## 输出格式

**必需文件：**
1. 设计哲学（.md 文件）
2. 视觉表达（.pdf 或 .png）

**哲学文件包含：**
- 运动名称
- 4–6 段哲学阐述
- 视觉原则
- 执行指引

**Canvas 文件包含：**
- 视觉诠释
- 极简文字
- 系统化构图
- 专家级执行
