# 论文筛选

## 适用范围

用于将候选文献池筛成可阅读、可引用、可进入综述的核心文献集合。

## 筛选维度

- 主题相关性：研究对象、变量、方法、场景是否匹配。
- 证据质量：同行评审、样本量、数据来源、实验或统计设计是否清楚。
- 文献类型：实证研究、综述、方法论文、理论论文、预印本。
- 时效性：优先近五年；保留领域奠基性经典文献。
- 可获取性：DOI、开放获取、Zotero 附件、PDF 或全文链接。

## 工作流

1. 读取用户目标与候选文献列表。
2. 明确纳入标准和排除标准；若用户未指定，先给出默认标准。
3. 按标题和摘要初筛，分为“纳入”“待定”“排除”。
4. 对待定文献进一步查看方法、结论、期刊或全文可得性。
5. 输出筛选表，并解释每篇文献的处理理由。

## 输出格式

```markdown
## 筛选标准
- 纳入：
- 排除：

## 纳入文献
- 文献：
  - 理由：
  - 用途：

## 待定文献
- 文献：
  - 需要补充的信息：

## 排除文献
- 文献：
  - 排除理由：
```

## 约束

- 不因标题相似直接判定重复，优先用 DOI、PMID、arXiv ID 与作者年份校验。
- 不用“高质量”代替具体评价，必须说明质量依据。
