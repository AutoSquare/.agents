# 视觉识别基础

## 核心视觉元素

### Logo
- **主版：** 完整 Logo（横版 / 竖版）
- **副版：** 缩写版
- **图标 / 标识：** 仅符号
- **安全区：** Logo 周围最小留白
- **最小尺寸：** 可辨识的最小尺寸

### 色板
```
Primary Colors (1-2)
├── Main brand color
└── Supporting primary

Secondary Colors (2-3)
├── Accent colors
└── Supporting visuals

Neutrals (3-4)
├── Text colors
├── Background colors
└── UI elements
```

### 字体
| 用途 | 字体 | 字重 | 字号 |
|------|------|------|------|
| H1 | [Font] | Bold | 32-48px |
| H2 | [Font] | Semibold | 24-32px |
| Body | [Font] | Regular | 16-18px |
| Caption | [Font] | Regular | 12-14px |

## 快速检查

### Logo
- [ ] 情境下使用正确版本
- [ ] 安全区充足
- [ ] 当前尺寸清晰可读
- [ ] 背景色搭配正确

### 色彩
- [ ] 仅使用批准色板
- [ ] 对比度满足无障碍要求
- [ ] 各物料应用一致

### 字体
- [ ] 使用指定字体
- [ ] 层级合理
- [ ] 字号可读
