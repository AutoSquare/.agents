# 品牌一致性清单

## 视觉一致性

### Logo
- [ ] 使用正确 Logo 版本
- [ ] 保持规定安全区
- [ ] 仅使用批准配色
- [ ] 各尺寸清晰可读
- [ ] 无未授权改动

### 色彩
- [ ] 仅使用品牌色板
- [ ] 配色应用一致
- [ ] 对比度满足无障碍要求
- [ ] 色彩比例符合规范

### 字体
- [ ] 使用品牌字体
- [ ] 字重与样式正确
- [ ] 层级清晰
- [ ] 格式统一

### 图像
- [ ] 符合品牌影像风格
- [ ] 后期 / 滤镜一致
- [ ] 题材合适
- [ ] 质量达标

## 调性一致性

### 语气
- [ ] 符合品牌人格
- [ ] 适配当前情境
- [ ] 各渠道表述一致
- [ ] 无相互矛盾的信息

### 用语
- [ ] 使用品牌术语
- [ ] 大小写规则一致
- [ ] 缩写规范统一
- [ ] 行话程度适当

### 信息
- [ ] 与核心信息对齐
- [ ] 价值主张清晰
- [ ] 差异化要点突出
- [ ] CTA 表述一致

## 渠道审计

### 网站
- [ ] 首页
- [ ] 产品页
- [ ] 博客 / 内容
- [ ] 页脚 / 导航

### 社交媒体
- [ ] 头像
- [ ] 封面图
- [ ] 简介 / 关于
- [ ] 帖子模板

### 邮件
- [ ] 页眉 / 页脚
- [ ] 模板
- [ ] 签名
- [ ] 自动化消息

### 宣传物料
- [ ] 演示文稿
- [ ] 单页资料
- [ ] 名片
- [ ] 促销物料

## 常见问题

| 问题 | 处理 |
|------|------|
| Logo 过时 | 替换为当前版本 |
| 偏离品牌色 | 更新为色板颜色 |
| 字体错误 | 替换为品牌字体 |
| 调性不一致 | 套用风格指南 |
| 信息混杂 | 对齐信息框架 |

## 审计频率

| 资产类型 | 频率 |
|----------|------|
| 网站 | 每月 |
| 社交账号 | 每季度 |
| 邮件模板 | 每季度 |
| 销售物料 | 每季度 |
| 全品牌审计 | 每年 |
