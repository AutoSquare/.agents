# 资产审批清单

营销资产批准前的完整审查清单。

## 快速审查

详细审查前确认：
- [ ] 资产符合既定用途
- [ ] 目标受众匹配
- [ ] 无明显错误或缺陷
- [ ] 与 campaign 目标一致

## 视觉元素

### Logo 使用
- [ ] 情境下使用正确变体
- [ ] 保持规定安全区
- [ ] 满足最小尺寸
- [ ] 仅使用批准配色
- [ ] 无未授权改动
- [ ] 与背景搭配合理

### 色彩合规
- [ ] 仅使用品牌色板
- [ ] 主 / 辅色比例合理（60/30/10）
- [ ] 语义色使用正确
- [ ] 未引入非品牌色
- [ ] 各元素配色一致

### 字体
- [ ] 全文使用品牌字体
- [ ] 字重应用正确
- [ ] 层级清晰
- [ ] 尺寸适配媒介
- [ ] 行高充足
- [ ] 正文无孤行 / 寡行

### 图像
- [ ] 符合品牌摄影风格
- [ ] 题材 / 内容合适
- [ ] 质量达标
- [ ] 授权 / 署名完备
- [ ] 已针对用途优化

## 无障碍

### 视觉无障碍
- [ ] 文字对比度 ≥ 4.5:1（AA）
- [ ] 大号文字对比度 ≥ 3:1
- [ ] 交互元素具备可见焦点
- [ ] 不单靠颜色传达含义
- [ ] 图像均有 alt 文本

### 内容无障碍
- [ ] 布局清晰可扫读
- [ ] 字号可读
- [ ] 阅读顺序合理
- [ ] 标题结构有意义
- [ ] 链接描述目标

## 内容质量

### 文案审查
- [ ] 符合品牌调性
- [ ] 语气适配情境
- [ ] 未使用禁用词
- [ ] 价值主张清晰
- [ ] CTA 明确有力
- [ ] 已校对无误

### 信息
- [ ] 与核心信息对齐
- [ ] 差异化要点突出
- [ ] 强调收益而非功能
- [ ] 面向目标受众
- [ ] 无相互矛盾表述

## 技术要求

### 文件规格
- [ ] 格式正确
- [ ] 分辨率适当
- [ ] 文件大小已优化
- [ ] 命名规范
- [ ] 元数据完整

## 常见问题与处理

| 问题 | 处理 |
|------|------|
| Logo 过小 | 增至最小尺寸 |
| 字体错误 | 替换为品牌字体 |
| 对比度不足 | 调整颜色以满足无障碍 |
| 非品牌色 | 替换为色板颜色 |
| 图像模糊 | 使用更高分辨率源文件 |
| 缺少 alt 文本 | 补充描述性 alt |
| CTA 薄弱 | 强化行动导向文案 |

## 自动化支持

`validate-asset.cjs` 脚本可自动检查：
- 色板合规
- 最小尺寸
- 文件格式 / 大小
- 命名规范
- 基础元数据

运行：`node "$env:USERPROFILE\.cursor\skills\brand\scripts\validate-asset.cjs" <asset-path>`

## 归档

批准后：
1. 在 manifest.json 中更新资产状态
2. 添加批准人与时间戳
3. 旧版本移入 archive
4. 更新 campaign 跟踪
5. 通知相关团队
