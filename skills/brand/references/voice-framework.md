# 品牌调性框架

## 调性与语气

**调性（Voice）** = 品牌人格（保持稳定）
**语气（Tone）** = 调性随情境的调整方式（可变）

示例：友好型品牌（调性）在庆祝公告中可更欢庆，在客服回复中则更共情（语气）。

## 调性维度

### 正式度谱系
```
Formal ←――――――――――――――→ Casual
[Legal docs]     [Social media]
```

### 语言复杂度谱系
```
Simple ←――――――――――――――→ Complex
[Consumer]       [Technical B2B]
```

### 性格谱系
```
Serious ←――――――――――――――→ Playful
[Finance]        [Entertainment]
```

### 情感表达谱系
```
Reserved ←――――――――――――――→ Expressive
[Corporate]      [Lifestyle brand]
```

## 调性制定流程

### 步骤 1：定义人格特质

选择 3 至 5 项描述品牌的特质：
- 自信，而非傲慢
- 友好，而非失礼
- 专业，而非居高临下
- 创新，而非噱头
- 真实，而非随意

### 步骤 2：建立调性对照表

| 特质 | 含义 | 应当 | 避免 |
|------|------|------|------|
| [Trait] | [Meaning] | [Example] | [Example] |

### 步骤 3：情境适配

| 情境 | 语气调整 | 示例 |
|------|----------|------|
| 社交媒体 | 更 casual | "Hey there!" |
| 客服支持 | 更共情 | "We understand..." |
| 法务合规 | 更 formal | "In accordance with..." |
| 销售转化 | 更 confident | "You'll see results..." |

## 调性检验

逐项确认：
1. 是否符合品牌调性？
2. 竞争对手是否会这样说？
3. 是否契合目标受众？
4. 是否与品牌价值观一致？
