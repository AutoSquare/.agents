# ppt-maker 发布说明

**版本**：ppt-maker-kit@0.2.0  
**发布载体**：`.agents/skills/ppt-maker/`（随 `.agents` 配置包分发）  
**用户安装**：`.agents/scripts/setup-cursor-agents.ps1` → `%USERPROFILE%\.cursor\skills\ppt-maker\`

---

## 能力概览

| 项 | 说明 |
|----|------|
| 输入 | 用户素材 + 结构化 `slides.js` |
| 预览 | Vue 16:9 分镜滚动页 |
| 导出 | 可编辑 PPTX、分层 ZIP、像素级 PPTX、幻灯片 ZIP |
| 风格 | 须 `grill-me` 顺序 intake + `ui-ux-pro-max` + `map-design-system` |
| 副本 | 仅写入工作区 `ppt-projects/{english-slug}/`，禁止改 `kit-template/` |

## 联动 Skills

- **grill-me**：开工前视觉定稿（Cursor 上 `AskQuestion` 每轮 1 题）
- **ui-ux-pro-max**：`search.py --design-system --persist`
- **academic-ppt-builder**：学术快出 pptx（答辩 / 开题 / 综述）

## 维护者（含本仓库沙箱时）

```text
编辑 ppt-maker/（工作区沙箱，可选）
  → node ppt-maker/scripts/sync-to-agents.mjs
  → cd .agents && powershell -File .\scripts\setup-cursor-agents.ps1
  → 重启 Cursor
```

**禁止**只改 `.agents/skills/ppt-maker/` 而不回写沙箱（若团队保留双树）。
