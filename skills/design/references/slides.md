# 幻灯片参考

战略性 HTML 演示文稿设计，集成 Chart.js 数据可视化、设计令牌、响应式版式与文案公式。

## 用法

激活 `design` 技能并指定幻灯片任务，例如「创建融资路演演示文稿」。

## 知识库

| 主题 | 文件 | 用途 |
|------|------|------|
| 创建指南 | `references/slides-create.md` | 分步幻灯片创建流程 |
| 版式模式 | `references/slides-layout-patterns.md` | 幻灯片版式模板与网格系统 |
| HTML 模板 | `references/slides-html-template.md` | 演示文稿基础 HTML 结构 |
| 文案公式 | `references/slides-copywriting-formulas.md` | 幻灯片内容的 AIDA、PAS、FAB |
| 幻灯片策略 | `references/slides-strategies.md` | 按演示类型的情境化策略 |

## 适用场景

- 营销演示与融资路演
- 集成 Chart.js 的数据驱动幻灯片
- 基于版式模式的战略性幻灯片设计
- 经文案公式优化的演示文稿内容
- 投资人路演、销售演示、团队汇报

## 核心能力

- **Chart.js 集成**：柱状图、折线图、饼图、环形图、雷达图
- **设计令牌**：统一的间距、色彩、字体
- **响应式**：适配桌面与移动端
- **文案公式**：内置 AIDA、PAS、FAB
- **版式模式**：Hero、分栏、网格、对比、时间线

## 工作流

1. 从用户请求解析演示类型
2. 加载 `references/slides-create.md` 获取创建指南
3. 从 `references/slides-layout-patterns.md` 选择版式
4. 应用 `references/slides-copywriting-formulas.md` 中的文案公式
5. 使用 `references/slides-html-template.md` 中的 HTML 模板
6. 应用 `references/slides-strategies.md` 中的策略
