# Logo 风格指南

## 核心 Logo 类型

### 1. Wordmark（文字标）
纯文字 Logo，采用定制字体。
- **最适合：** 成熟品牌、易记名称
- **示例：** Google、Coca-Cola、FedEx
- **字体：** 定制字形、独特字距
- **提示：** 名称须易记、易读

### 2. Lettermark（字母标）
首字母或缩写。
- **最适合：** 长公司名、专业机构
- **示例：** IBM、HBO、NASA
- **字体：** 粗体几何无衬线
- **提示：** 适合 2–4 字母缩写

### 3. Pictorial Mark（图形标）
独立图标或符号。
- **最适合：** 已有认知的全球品牌
- **示例：** Apple、Twitter、Target
- **设计：** 简洁、可缩放、易记形状
- **提示：** 需品牌积淀方可单独使用

### 4. Abstract Mark（抽象标）
非具象几何形状。
- **最适合：** 科技公司、差异化品牌
- **示例：** Pepsi、Airbnb、Spotify
- **设计：** 独特形状传达品牌价值
- **提示：** 可简洁表达复杂理念

### 5. Mascot（吉祥物）
代表品牌的角色形象。
- **最适合：** 家庭品牌、体育、餐饮
- **示例：** KFC、Pringles、Michelin
- **设计：** 友好、富有表情、可延展
- **提示：** 可随品牌演进仍保持识别度

### 6. Emblem（徽章标）
符号置于封闭形状内。
- **最适合：** 传统品牌、组织
- **示例：** Starbucks、Harley-Davidson、NFL
- **设计：** 徽章、印章或纹章风格
- **提示：** 小尺寸缩放可能受限

### 7. Combination Mark（组合标）
图标 + 文字统一设计。
- **最适合：** 新品牌、多场景应用
- **示例：** Burger King、Lacoste、Doritos
- **设计：** 锁定版式，排列灵活
- **提示：** 最灵活，后期可拆分元素

## 美学风格

### Minimalist（极简）
- 干净线条，仅保留必要元素
- 大留白，简单几何
- 色板有限（1–2 色）
- **适用：** 科技、专业服务、现代品牌

### Vintage/Retro（复古）
- 怀旧、传承感
- 做旧纹理、柔和色
- 手写体或粗衬线
- **适用：** 手工艺品牌、 artisan 产品

### Luxury/Premium（奢华）
- 优雅、精致
- 金、黑、白配色
- 细衬线或精致无衬线
- **适用：** 时尚、珠宝、高端服务

### Geometric（几何）
- 数学精确
- 圆、三角、方
- 完美对称
- **适用：** 建筑、科技、现代品牌

### Organic/Natural（有机）
- 流动、不完美线条
- 大地色、自然色
- 手绘感
- **适用：** 环保、健康、有机产品

### Gradient/Modern（渐变/现代）
- 色彩过渡、鲜明色板
- 立体感
- 当代感
- **适用：** 应用、科技创业、数字产品

## 风格选择矩阵

| 品牌类型 | 主风格 | 备选 |
|----------|--------|------|
| 科技创业 | Minimalist、Abstract | Geometric、Gradient |
| 律所 | Wordmark、Emblem | Lettermark |
| 餐饮 | Mascot、Badge | Vintage、Combination |
| 时尚 | Wordmark、Luxury | Monogram、Line Art |
| 医疗 | Professional、Line Art | Abstract、Combination |
| 非营利 | Combination、Emblem | Organic、Hand-Drawn |

- [ ] 16×16 像素（favicon）仍可识别
- [ ] 名片尺寸清晰
- [ ] 可单色呈现
- [ ] 黑白仍清晰
- [ ] 缩放时无细小细节消失
