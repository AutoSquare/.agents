# 研究空白分析

## 适用范围

用于把一组核心文献转化为选题判断、综述结构和后续检索路线。

## 工作流

1. 按研究对象、理论视角、方法、数据、场景和结论对文献分组。
2. 提炼领域共识：哪些问题已被反复验证。
3. 提炼争议点：结论冲突、方法差异、样本差异或指标定义差异。
4. 提炼空白：对象不足、场景不足、方法不足、数据不足、理论解释不足。
5. 将空白转化为可研究问题，说明可行性和证据基础。

## 输出格式

```markdown
## 研究脉络
- 阶段/主题：
  - 代表文献：
  - 主要观点：

## 共识与争议
- 共识：
- 争议：

## 研究空白
1. 空白：
   - 依据：
   - 可研究问题：
   - 可行性：

## 后续检索路线
- 关键词：
- 数据源：
- 应补充的文献类型：
```

## 约束

- 研究空白必须由文献证据支持，不能只来自常识推断。
- 不把“文献少”直接等同于“有研究价值”，必须说明理论或实践意义。
