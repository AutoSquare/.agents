## 22 种美术指导风格

1. **Minimalist** — 留白为主，单一焦点，1–2 色，简洁无衬线
2. **Bold Typography** — 字体即设计；超大、富有表现力的字形填满画布
3. **Gradient / Color Wash** — 平滑过渡、网格渐变、色彩混合
4. **Photo-Based** — 全出血摄影叠加文字；生活方式 Hero 图
5. **Illustrated / Hand-Drawn** — 定制插画、专属图标、手工艺感
6. **Geometric / Abstract** — 形状、线条、网格为主要视觉元素
7. **Retro / Vintage** — 做旧纹理、柔和色板、衬线、半调网点
8. **Glassmorphism** — 磨砂玻璃面板、背景模糊、细边框光晕
9. **3D / Sculptural** — 渲染物体、景深、阴影；以产品为中心
10. **Neon / Cyberpunk** — 深色背景、霓虹高光、高对比
11. **Duotone** — 双色照片处理；品牌色叠加于图像
12. **Editorial / Magazine** — 网格密集、引语、杂志式构图
13. **Collage / Mixed Media** — 剪纸纹理、照片裁切、层叠元素
14. **Retro Futurism** — 太空时代怀旧、镀铬、渐变、乐观感
15. **Expressive / Anti-Design** — 混乱版式、混用字体、刻意「错误」构图
16. **Digi-Cute / Kawaii** — 圆角、粉彩渐变、像素风、可爱角色
17. **Tactile / Sensory** — 蓬松/柔软质感、超写实材质、浮雕感
18. **Data / Infographic** — 数据居中，图表、数字为视觉主角
19. **Dark Mode / Moody** — 近黑背景、浓郁宝石色、高对比
20. **Flat / Solid Color** — 单色背景、简洁图标、无渐变
21. **Nature / Organic** — 大地色调、植物纹样、可持续品牌感
22. **Motion-Ready / Kinetic** — 为动画设计；分层元素、可循环

## 设计原则

### 视觉层级（三区法则）
- **上部**：Logo 或核心价值主张
- **中部**：支撑信息 + 视觉
- **下部**：CTA（按钮/二维码/URL）

### 安全区
- 关键内容置于画布中央 70–80%
- 文字/CTA 距边缘 50–100px 以外
- YouTube：在 2560 × 1440 内使用 1546 × 423px 安全区
- Meta/Instagram：中央 80% 以避免 UI 遮挡

### CTA 规则
- 每幅横幅一个 CTA
- 与背景高对比
- 右下放置（终端区）
- 移动点击目标最小高度 44px
- 行动动词：Get、Start、Download、Claim

### 字体
- 每幅横幅最多 2 种字体
- 数字端：正文最小 16px，标题 ≥32px
- 对比度最低 4.5:1
- 广告每行最多 7 词、最多 3 行

### 图文比例
- 广告：文字低于 20%（Meta 会降权）
- 社交封面：图 60 / 文 40
- 印刷：3–5 米视距标题 70pt+

### 印刷规格
- 最低 300 DPI（大幅面可用 150 DPI）
- 四边出血 3–5mm
- CMYK 色彩模式
- 视距规则：每英尺 1pt
